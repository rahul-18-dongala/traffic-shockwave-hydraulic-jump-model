# traffic-shockwave-hydraulic-jump-model
# Hydraulic Jump Lab: Modeling Traffic Shockwaves

## Overview
This project models traffic shockwaves using the hydraulic jump analogy.

Traffic flow is simulated using SUMO and analyzed using MATLAB.

## Tools Used
SUMO (Simulation of Urban Mobility)
MATLAB
Python (optional)

## Methodology
1. Road network created in SUMO
2. Traffic congestion generated
3. Shockwave formation observed
4. MATLAB used for analysis

## Results
Shockwave speed calculated
Density vs time graphs generated

## Applications
Traffic congestion analysis
Highway traffic modeling
Transportation engineering research
